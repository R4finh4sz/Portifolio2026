## ✨ Sobre

Portfólio desenvolvido para apresentar minha trajetória como desenvolvedor Fullstack, projetos reais e habilidades técnicas. Design dark com animações fluidas, glassmorphism e gradientes modernos.

---

## 🛠️ Stack

| Tecnologia | Uso |
|---|---|
| **Next.js 16** | Framework React com SSG |
| **TypeScript** | Tipagem estática |
| **Tailwind CSS 4** | Estilização utilitária |
| **Motion (Framer)** | Animações e transições |
| **Lucide React** | Ícones |

## 📁 Estrutura

```
src/
├── app/              # Layout e página principal (Next.js App Router)
├── components/
│   ├── Hero/         # Seção inicial com apresentação
│   ├── About/        # Seção sobre mim com foto
│   ├── Skills/       # Grid de tecnologias
│   ├── Projects/     # Projetos do GitHub
│   ├── Contact/      # Seção de contato
│   ├── Nav/          # Navegação
│   └── FuturisticBg/ # Background animado
├── styles/           # CSS global, tema e fontes
└── Assets/           # Foto de perfil
```

## 🚀 Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/R4finh4sz/Portifolio2026.git
cd Portifolio2026

# Instale as dependências
yarn install

# Inicie o servidor de desenvolvimento
yarn dev
```

Acesse: [http://localhost:3000](http://localhost:3000)

---

## 🏗️ Build para produção

```bash
yarn build
yarn start
```
